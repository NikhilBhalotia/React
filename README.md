## 🚀 Getting Started

1. **Clone this repository**:
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git
   cd <repo-name>
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```
   Access the app at `http://localhost:5173/` (Vite) or `http://localhost:3000/` (CRA).
